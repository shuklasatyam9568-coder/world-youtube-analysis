# world-youtube-analysis
# 📊 World YouTube Data - Basic EDA Project

This project performs **Basic Exploratory Data Analysis (EDA)** on a World YouTube dataset using Python and Pandas.

The main objective is to understand the dataset, inspect data, clean columns, and analyze YouTube growth trends across different countries using socio-economic and YouTube-related metrics.

---

## 🚀 EDA Steps Performed
- Load dataset
- Basic inspection of data
- Check shape, columns, and data types
- Rename columns
- Reset index
- Handle missing values
- Boolean indexing and filtering
- Indexing and slicing
- Unique value analysis
- Value counts and category analysis

---

## 🧠 Case Study 1: Analyzing YouTube Growth Trends Across Countries

### 📌 Questions Solved

### 1. Reading & Boolean Indexing
- Filter all YouTubers from India
- Unemployment rate below 6%
- More than 5 million subscribers

### 2. Indexing & Slicing + Unique
- Display first 5 YouTubers using slicing
- Find unique channel types in dataset

### 3. Value Counts + Boolean Indexing
- Identify top 3 most common YouTube categories
- Count number of YouTubers in the Entertainment category
